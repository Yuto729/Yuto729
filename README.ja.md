## About

東京大学精密工学科（2026年4月より大学院進学予定）のソフトウェアエンジニアです。バックエンド・AI統合・データパイプラインを中心に、信頼性とビジネス価値を意識した開発をしています。

AI時代に本当に価値があるのは、コードを書くことではなく「**正しい問いを立てること**」だと思っています。問題を構造化し、確かな意思決定と責任を持って実装まで自走することを大切にしています。

- 🎓 東京大学 精密工学科 → 工学系研究科 精密工学専攻（2026年〜）
- 🏢 現在：株式会社Bizgem フルスタックエンジニア
- 🔬 研究：IMUを用いた歩行解析による体重変化推定
- 🤖 松尾研TA（世界モデル・深層学習基礎）
- 📍 東京都
