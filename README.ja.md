## About

東京大学工学系研究科に在籍中の学生エンジニアです。バックエンド・インフラ・LLMを軸足としてシステムの信頼性とビジネス価値を意識した開発をしています。

AI時代に大切なことは「**何をしたいのか？何のために手を動かすのか**」を常に考えることだと思っています。問題を構造化し、確かな意思決定と責任を持って実装まで自走することを大切にしています。
